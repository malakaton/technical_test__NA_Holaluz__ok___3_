Docker PHP-NGINX
==============

# Requirements

To use this docker-compose.yml, you will need:

- Docker engine > 19.03
- docker-compose > 1.26

Both are available in the [Docker official site](https://docs.docker.com/install/)_. All tests were performed with Docker CE.

# Installation

 To run, execute:

```bash
$ docker-compose up --build -d
```

If you want close docker compose, run this:

```bash
$ docker-compose down
```

# Show results

Visit localhost:8080 in your browser to see the results of exercise. Example: http://localhost:8080/matrices.php
